# pause - угадай

qq - старые эти самые (их можно включить по цифре)

remove, rm - убрать песню из очереди по номеру (например 1 3 5-15 17)
rmlist - посмотреть че убрали
clear, clean, cc - очистить очередь

pp - включить локальный плейлист по названию или по ссылке с ютуба (по ссылке это тебе не локальный)
save, ss - сохранить локальный плейлист по названию (save мама не бей)
